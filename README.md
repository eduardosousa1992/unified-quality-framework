## 🚀 1. Narrative & Business Problem
### **The Problem**
Sistemas financeiros operam com volumes massivos de dados. Validações manuais geram riscos de conformidade.
### **The Solution**
O **UQF** automatiza a governança de dados via regras configuráveis e gera um **Quality Score** para tomada de decisão.
