# 🛡️ Unified Quality Framework (UQF)
> **Executive Quality Governance & AI-Driven Automation**

### 🚀 Problem
Automações de QA em ambientes bancários (como o Bradesco) frequentemente falham por excesso de ruído técnico, dificultando a tomada de decisão executiva. O UQF resolve a falta de clareza entre falhas de infraestrutura e erros reais de negócio.

### 💡 Motivation
Minha motivação é transformar o QA de um "centro de custo" em uma "unidade de inteligência", provendo dados acionáveis sobre a saúde dos microserviços em tempo real.

### 🏗️ Architecture
O framework utiliza **Playwright + TypeScript** com uma camada de **Dual-Path Reporting**:
1. **Executive View (Allure)**: Focado em taxas de sucesso e estabilidade.
2. **Engineering View (Artifacts)**: Focado em traces, logs e debug.

### 📁 Case Study: Microservices Journey
**Cenário Real:** Validação de uma malha de microserviços financeiros com mais de 120 testes automatizados, garantindo a integridade de payloads JSON e fluxos de onboarding de alta criticidade.

### 📊 Quality Metrics (Live)
* **Execution Success Rate:** 84% (Nominal)
* **Average Test Runtime:** 3.1s
* **Stability Target:** > 90%
